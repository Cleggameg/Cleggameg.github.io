Enumerable Methods

January 18, 2015

Enumerables are power tools that are used on collections. This could be a range, an array, or a hash. What they do is go through that collection and perform an action to each thing within the collection. A great way to explain this would be using the enumerable .map

.map is great because it allows you to perform an action to every single element within your collection, and output those in the form of a new array. Letâ€™s take a look:

a = [1, 2, 3, 4]
a.map { |x| x*2 } =>[2, 4, 6, 8]

The changes that you make to map do not modify the original collection. If you want to do that you would have to use the .map! command. It works similarly, except the changes are permanent:

a = [1, 2, 3, 4]
a.map { |x| x*2 } =>[2, 4, 6, 8]
a =>[1, 2, 3, 4]
a.map! { |x| x*2 } =>[2, 4, 6, 8]
a =>{2, 4, 6, 8]
