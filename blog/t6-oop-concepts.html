OOP vs. FP

January 30, 2015

In programming two of the major schools of thought are Object Oriented Design (OOD) and Functional Programming (FP).

OOD revolves around the creation and use of objects. Within OOD, an object owns its data and it is responsible for methods called on that data. OOD revolves around the creation of new objects and running new or existing methods on those objects. Objects are define by the data they contain. Objects interact with by the methods we call to them.

On the other end FP revolves around functions. In it, you can write different operations for the same data structures. In FP, programming is done through combining and expressing functions as opposed to objects with OOD.
