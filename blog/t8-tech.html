What is rails?

February 15, 2015

Rails or Ruby on Rails is a software package that works with and extends the ruby language. It is a framework that can be used to build websites quickly. As such, it is the framework of choice for many startups. It encourages you to program through convention over configuration and emphasizes the donâ€™t repeat yourself rule of programming. It allows for speedy development because the redundancy of many coded applications is reduced. Sticking to convention will also allow you to increase efficiency by allowing your application to make certain assumptions about the code. It seems like rails is all about efficiency.

Rails has several components to it: Active record, Action pack, Action mailer, Action web, and Prototype. All these components help to make working rails easier. It supports many SQLs.
