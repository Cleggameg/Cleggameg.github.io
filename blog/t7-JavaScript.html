Loops in Rails and JavaScript

February 7, 2015

Loops within Ruby and JavaScript are fairly similar. the for and .times loops both work fairly similarly. They’re both used to do something a set number of times. In each of the two examples below I use some sort of counter variable. Each one prints the counter to the screen and each one ends when the counter hits 4.

for (x=0; x <5; x++){ 
console.log(x); 
};

x = 0
5.times do
puts x
x = x + 1
end

While loops both act similarly as well. Their syntax is only slightly different. Both these run a block of code until the condition is met

while (condition) {
code
}

while condition [do]
code
end
